# 🐉 青龙面板日常签到小助手 (Qinglong-Daily-Scripts)

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Qinglong](https://img.shields.io/badge/青龙面板-支持-success.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

这是一个专为 [青龙面板](https://github.com/whyour/qinglong) 打造的日常自动化脚本库。主打**极简、稳定、防封**。
全部脚本基于纯 Python 编写，无需配置复杂的 Node.js 环境。

## 🌟 核心特色

* **零额外依赖**：只需安装最基础的 `requests` 库即可完美运行。
* **云端防风控**：部分高难度加密接口（如网易云）已剥离至第三方稳定云节点，彻底告别本地加解密报错。
* **消息精美推送**：内置 Bark 精美排版推送逻辑，运行结果一目了然。

### 1. 📺 B站日常助手 (`bili_task.py`)

## 🚀 青龙面板部署指南

### 第一步：安装依赖
进入青龙面板 -> **依赖管理** -> **Python3**，添加并安装以下依赖：
    
    requests

### 第二步：添加订阅 (拉库)
进入青龙面板 -> **订阅管理** -> 新建订阅：
* **名称**：`日常签到小助手`
* **类型**：`公开仓库`
* **链接**：`https://github.com/Han-cy77/Qinglong_auto.git`  
* **定时规则**：`0 6 * * *` （每天早上 6 点自动同步最新代码）

保存后，点击运行该订阅即可自动生成任务。

## ⚙️ 环境变量配置

请在青龙面板的 **环境变量** 中添加以下变量，才能让脚本认出你的账号：

| 变量名称 | 必须 | 说明与获取方式 |
| :--- | :---: | :--- |
| `BILI_COOKIE` | 否 | **B站通行证**。网页端登录后 F12 抓取 `bilibili.com` 的完整 Cookie。 |
| `NETEASE_COOKIE` | 否 | **网易云通行证**。网页端抓取，格式必须为：`MUSIC_U=你的值; __csrf=你的值;` |

*(注：你跑哪个脚本，就配哪个变量即可。不跑的无需配置。)*

## 🔔 推送配置说明

本项目默认内置苹果 **Bark** 推送接口。
如果你需要开启推送，请在拉库成功后，打开青龙面板的 **脚本管理**，找到对应的 `.py` 文件，将文件顶部的 `BARK_URL` 替换为你自己的专属链接即可：

    # ================= 配置区域 =================
    BARK_URL = 'https://api.day.app/你的专属Key'
    # ============================================
